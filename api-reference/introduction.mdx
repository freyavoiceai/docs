## Overview

The Freyavoice Call API under `/api/v2/call` powers call search, details, recordings, tags, and problems. The Mintlify API playground below is generated from our OpenAPI spec.

## Key endpoints

- List calls: `GET /call` (filters, pagination, export)
- Call details: `GET /call/{callId}` (transcript, analysis, feedback, tags, problems)
- Tags: `POST|PATCH|DELETE /call/{callId}/tags`
- Problems: `GET|POST|PATCH|DELETE /call/{callId}/problems`
- Recording: `GET|POST /call/{callId}/recording`
- Overview metrics: `GET /call/overview`

## Authentication

All endpoints require a bearer token issued by Freyavoice. In OpenAPI this is represented as:

```json
"security": [
  {
    "bearerAuth": []
  }
]
```
